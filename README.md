# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.  
This project was created to practice frontend web development concepts like DOM manipulation and event handling


🔗 Play the game here:  
https://tictactoe-byhamza-12.surge.sh/


---

## 📌 Features

- ✅ Two-player Tic Tac Toe gameplay
- ✅ Interactive game board
- ✅ Winner detection
- ✅ Draw detection
- ✅ Reset game button
- ✅ New game option
- ✅ Clean and simple user interface

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the game
- **CSS3** –   For  Styling 
- **JavaScript** – Game logic and interactions
